# Day 1 - Introduction to RAG

Today I learned about Retrieval Augmented Generation (RAG).

## What is RAG?
RAG is an AI architecture that combines:
- Retrieval systems
- Large Language Models (LLMs)

It helps AI answer questions using external documents instead of relying only on training data.

## Main Components
- Embedding Model
- Vector Database
- Retriever
- LLM

## Tools I Explored
- LangChain
- FAISS
- LLaMA 3

## Learning Outcome
Understood how AI chatbots can answer questions from custom documents.
