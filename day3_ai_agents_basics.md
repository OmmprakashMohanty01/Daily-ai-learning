# Day 3 - AI Agents Basics

## What are AI Agents?

AI Agents are systems that use Large Language Models (LLMs) together with tools, memory, and reasoning to perform tasks autonomously.

Unlike traditional chatbots that only generate text, AI Agents can:

* Use external tools
* Search information
* Make decisions
* Execute multi-step workflows
* Interact with APIs

## Key Concepts Learned

### 1. LLM as the Brain

The LLM performs reasoning and planning.

Examples:

* GPT
* Llama
* Claude
* Gemini

### 2. Tools

Agents can use tools such as:

* Search engines
* Databases
* Calculators
* APIs

### 3. Memory

Memory allows agents to remember previous interactions and context.

### 4. Planning

Agents can break complex tasks into smaller steps and execute them sequentially.

## Real-World Applications

* Customer Support Automation
* Research Assistants
* Workflow Automation
* AI Coding Assistants
* Knowledge Management Systems

## Key Takeaway

AI Agents extend the capabilities of LLMs by combining reasoning, memory, and tool usage to solve complex tasks more effectively.
