# Day 2 - Introduction to Prompt Engineering

Today I explored the basics of Prompt Engineering.

## What is Prompt Engineering?

Prompt Engineering is the practice of designing clear instructions that help AI models generate better outputs.

## Key Learnings

### 1. Be Specific
Instead of:
"Write about AI"

Use:
"Explain AI to a beginner in 5 bullet points."

### 2. Provide Context
Giving background information helps the model understand the task better.

### 3. Define the Output Format
Example:
- Bullet points
- Table
- Step-by-step guide
- JSON

## Example Prompt

Explain Retrieval Augmented Generation (RAG) in simple terms.
Provide:
1. Definition
2. Components
3. Real-world example

## Learning Outcome

Understood how better prompts can improve AI responses and reduce ambiguity.
